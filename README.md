📝 Gerenciador de Tarefas Pessoal
Aplicação web simples para gerenciar tarefas pessoais, desenvolvida com Flask e JSON como banco de dados.

🚀 Funcionalidades

✅ Adicionar novas tarefas
✅ Marcar tarefas como concluídas
✅ Reabrir tarefas concluídas
✅ Deletar tarefas
✅ Filtrar por status (todas, pendentes, concluídas)
✅ Dados persistidos em arquivo JSON


🛠️ Tecnologias utilizadas

Python 3
Flask
Jinja2
HTML e CSS puro


📁 Estrutura do projeto
gerenciador-tarefas/
├── app.py              # Rotas e lógica da aplicação
├── tarefas.json        # Banco de dados local (gerado automaticamente)
├── templates/
│   ├── base.html       # Layout base
│   └── index.html      # Página principal
└── static/
    └── style.css       # Estilos da aplicação

▶️ Como rodar o projeto
Pré-requisitos
Python 3 instalado
Git instalado
